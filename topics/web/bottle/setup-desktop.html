Tool to install and configure Tableau Desktop
 -b,--branch <name>                 Branch installer, e.g. '-b near'. Use the latest unless build is specified.
    --build <number>                Build number, e.g. 15.0829.2025. Used with branch option.
    --buildType <type>              Specify a build type, NIGHTLY, CONTINUOUS, or REMOTE. Default is all build types if
                                    a build number is specified, or official builds only (nightly or continuous) if
                                    looking for the latest build of a branch.
    --desktopInstaller <filename>   Desktop installer filename, e.g. myDesktop.exe. Used with 'installerDir' option
                                    tospecify a non-default name
 -ds,--desktopSku <sku>             Specify the type of desktop, [STD or PUBLIC or ONLINE or READER]; defaults to STD
 -h,--help                          Print help
 -i,--installerDir <dir>            Private build installer directory. Should provide the custom installer name unless
                                    file name option is used
 -id,--installDir <dir>             Complete path to installation directory (where desktop binaries will be copied).
                                    Overrides the default path for install
 -n,--nodes <target machine>        Remote machine name to install desktop on
 -p,--password <value>              password for the user on remote machine, override the default workgroupuser
                                    password, used with target option
 -q,--quiet                         Don't output log messages
    --rc <number>                   Release candidate number, e.g. 6 (for rc6). Used with 'version' option
 -t,--target <target machine>       Remote machine name to install desktop on
 -u,--username <value>              username on remote machine, override the default of workgroupuser, used with target
                                    option
 -uo,--uninstallonly                Uninstalls the Desktop. Supports clean and non default location flags during
                                    uninstall
 -v,--version <number>              Version installer, e.g. 9.0.2. Use the latest unless rc is specified
    --x86                           Specify 32-bit installer.
